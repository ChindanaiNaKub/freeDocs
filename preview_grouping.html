XML Grouping Improvement Demo

After Improvement: Grouped Dependencies (Fewer Boxes)

Each complete dependency is now grouped into a single logical block:

<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <version>1.18.34</version>
    <scope>provided</scope>
</dependency>
+ <dependency>
+     <groupId>org.springframework</groupId>
+     <artifactId>spring-core</artifactId>
+     <version>5.3.21</version>
+ </dependency>
- <dependency>
-     <groupId>junit</groupId>
-     <artifactId>junit</artifactId>
-     <version>4.12>
- </dependency>

Summary of Improvement

✅ Before: 1 dependency = 6 separate boxes (one for each XML tag)
✅ After: 1 dependency = 1 logical box (all related tags grouped together)
✅ Result: Much cleaner, easier to read, and more intuitive presentation
